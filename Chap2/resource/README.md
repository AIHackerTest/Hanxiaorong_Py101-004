### 选择合适的天气 API

了解什么是 API 后，是时候选择一个顺手的天气 API 来完成本章任务啦~

弱水三千，只取一瓢。互联网上有非常多的天气 API，那如何选择一个合适的 API ？

对于本任务，建议你在选择 API 时考虑以下问题：

是否有 Python 的 demo ？
使用上有哪些限制（比如要收费）？
数据源是否可靠？
...
